# **App Name**: Football Banter Barn

## Core Features:

- Google Sign-In: Authenticate users using Google Sign-In for a seamless experience.
- Match Stats Extraction: Extract match statistics from uploaded images using the Gemini API tool.
- Duplicate Detection: Identify duplicate match uploads using SHA-256 hash comparison.
- Roast Generation: Generate funny movie quotes and savage roast messages based on match stats using the Gemini API tool.
- Leaderboard Calculation: Calculate and display all-time, daily, and monthly leaderboards using Firestore.
- User Profile: Create and manage user profiles with stats, achievements, and match history.
- Hall of Shame: Display categories of embarrassing match stats and roast the 'winners'. Generate the roasts and troll categories using an AI model tool
- Match Listing and Filtering: Allow users to view and filter their own matches, as well as other users' matches, by score, win/loss, and other criteria.
- Match Commenting: Enable users to add comments to matches for trash talk and banter.
- Dynamic Badge Generation and Assignment: Generate different badges with descriptions using AI. When a user uploads a match, check for badge criteria and assign badges dynamically. Other users can view these badges.
- Rivalry System: Automatically detect rivals (most played against opponent) based on match history.
- Opponent Analysis: Provide analysis of opponents, highlighting strengths and weaknesses, and display the user's form guide (last 5 matches).
- Career Milestones: Track progress towards career milestones (e.g., 100 wins, 500 goals), celebrate milestones with animations, and allow users to share milestone cards. Provide a 'Journey' page showing progress over time.
- Advanced Badges: Implement advanced badges such as 'Comeback King,' 'Giant Killer,' 'Consistent,' 'Marathon Runner,' and 'Late Game Hero' based on specific in-game achievements.

## Style Guidelines:

- Primary color: Yellow (#e7e600) for wins, success, and key interactive elements, conveying a sense of energy and excitement.
- Secondary color: Dark blue (#000064), creating a sophisticated, sporty, and high-tech feel. The low saturation keeps the color from distracting.
- Third color: Light blue (#0099f8) to highlight important stats, match details, and create visual interest, complementing the primary color with a similar futuristic vibrancy.
- Headings font: 'Bebas Neue', sans-serif, for bold and sporty headings to match the theme.
- Body font: 'Inter', sans-serif, for clean and readable text.
- Use dynamic icons to indicate wins, losses, or draws.
- Incorporate smooth animations for page transitions, confetti explosions on win submissions, and hover effects on interactive elements to enhance user experience.
